# PUSL3120 Fullstack Coursework Ecommerse Site

Laptop.LK

## Running the Application

### Docker



```bash
docker compose up
```
- api will be hosted on port : 3001
- frontend web application will host on port : 3000
- database will be connected to mongodb online(network connection required)


### without docker

setup frontend and backend separately without docker

### seting up backend

```bash
cd backend
npm install
npm run dev
```


### seting up frontend

```bash
cd frontend
npm install
npm run dev
```



### Running test cases


### backend test cases

```bash
cd backend
npm install
npm run test
```


### frontend test cases

```bash
cd frontend
npm install
npm run test
```
